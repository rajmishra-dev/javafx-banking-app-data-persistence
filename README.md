# JavaFX - Create Banking Application With Data Persistence - 2025

This repository contains a **JavaFX-based Banking Application** built by following the YouTube tutorial:  
**"JavaFX - Create Banking Application With Data Persistence - 2025"**.

The app demonstrates:
- Modern JavaFX UI
- Basic banking operations (accounts, deposits, withdrawals, transfers)
- Data persistence using a relational database
- Clean separation between UI, business logic, and data access
